# **App Name**: TheCanIndian

## Core Features:

- Authentication Page: Landing page with login/signup forms, including social login (Google, LinkedIn) and a hero image.
- Main App Layout: Sidebar navigation for key pages (Draw Tracker, Dashboard, Profile, Program Tracker, Chatbot, FAQ).
- Dashboard: Dashboard to show key stats, charts for user eligibility and PR chances.
- Program Tracker: Form-based input for program eligibility checks, with results displayed in a card or table format.
- AI Chatbot: AI-powered chatbot tool for immigration queries, featuring highlighted AI responses and persistent chat history.
- FAQ: FAQ page with an accordion-style, searchable list of common questions.
- Responsive Design: Responsive design ensuring seamless operation across desktop, tablet, and mobile devices.

## Style Guidelines:

- Primary color: Darker desaturated violet (#4B0082) to maintain the sophisticated and reliable aspects of a modern service.
- Background color: Deep charcoal (#121212) for the dark mode theme.
- Accent color: Vivid yellow-green (#BFFF00) for highlighting interactive elements and important information.
- Body and headline font: 'Inter', a grotesque-style sans-serif known for its modern and neutral aesthetic. It ensures readability and a clean look.

- Subtle loading animations and hover effects on buttons and cards enhance the user experience without being intrusive.
